This program is a evaluation tool for the pokemons. It output the pokemon with favourable type advantage. If it is a type advantage tie, the pokemon with greatest total stat will output.


How to run:
 ./pokemon name#1 name#2 ...


Note:
 1. input can be pokemon's ID or pokemon's name.
 2. Pokemon's name can be capital letter or lowcase letter.
 3. Pokemon's ID have to be the number of 1-802  (2017/11/6)

used API: https://pokeapi.co/api/v2/
